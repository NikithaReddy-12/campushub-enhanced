# CampusHub

CampusHub is a full-stack student engagement platform designed to simplify campus activities and communication.

## Features

* Club Management
* Event Creation and Registration
* Student Authentication
* Notifications System
* Event Tracking
* Role-Based Access Control

## Tech Stack

### Frontend

* HTML
* CSS
* JavaScript

### Backend

* Node.js
* Express.js

### Database

* MongoDB
